# Playwright Project

## Setup

1. Installer les dépendances :

\`\`\`
npm install
npx playwright install
\`\`\`

2. Configurer le fichier \`.env\` avec les bonnes variables d'environnement

3. Lancer les tests :

\`\`\`
npm test
\`\`\`

4. Générer et afficher le rapport :

\`\`\`
npx playwright show-report
\`\`\`
